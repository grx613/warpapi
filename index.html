Примеры интеграции

JS · Чат

OpenAI SDK 
import OpenAI from "openai";

const client = new OpenAI({
  apiKey: "sk-warpapi-xxx",
  baseURL: "https://api.warpapi.ru/v1",
});

const resp = await client.chat.completions.create({
  model: "deepseek-r1",
  messages: [{ role: "user", content: "Дай интересный факт" }],
  max_tokens: 50000 // задайте явным параметром
});

console.log(resp.choices[0]?.message);

Python · Чат

OpenAI SDK 
from openai import OpenAI

client = OpenAI(
    api_key="sk-warpapi-xxx",
    base_url="https://api.warpapi.ru/v1",
)

resp = client.chat.completions.create(
    model="gpt-5",
    messages=[{"role": "user", "content": "Привет! Что ты умеешь?"}],
    max_tokens=2048
)

print(resp.choices[0].message)

FAQ

Работаете с юрлицами? Нужны документы.
Да. Счёт на оплату, акт/УПД, ЭДО и индивидуальные условия при больших объёмах.
Что такое max_tokens и почему важен?
max_tokens ограничивает длину ответа. Мы резервируем средства по ожидаемому максимуму — это помогает предсказать стоимость. Для reasoning‑моделей (например, r1/o‑серии) выставляйте не меньше ~50 000, чтобы не получить пустой ответ.
Есть ли лимиты по запросам?
Лимиты масштабируются от вашего тарифа/баланса. При необходимости повышаем: напишите нам.
Можно ли использовать в n8n / LangChain?
Да. Выберите OpenAI, укажите наш токен и baseURL https://api.warpapi.ru/v1. В LangChain — задайте OpenAICompatible client.
